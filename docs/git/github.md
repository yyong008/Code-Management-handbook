# github

## github https

`https://github.com/yyong008/ts-translate.git`

- 固定协议： https
- 固定域名： github.com
- 用户注册名：yyong008
- 项目名称：ts-translate

## git ssh

`git@github.com:yyong008/ts-translate.git`

- 头：`git`
- @内容：`@github.com`
- 分割符：`:`
- 用户名：`yyong008`
- 项目名：`ts-translate`
- git文件：`.git` 我们知道git的文件全部放在 `.git 文件夹下`
